Hello, I'm Raymond

I am a web developer based in East London and I love create great looking web applications and websites that provide effective solutions to your online needs.

I design, construct and deploy websites and web applications that follow the latest web standards and best practices, are performant, secure and can be viewed on most browsers and mobile devices.

I also create other cool stuff for the web.

About Me

I became interested in programming and web development after attending a part-time web design course in 2012.

I have a background in Local Government Revenues and Benefits in assessment and customer service but found programming and the whole web development process fascinating. I spent lots of my spare time teaching myself code and started designing, constructing, deploying, maintaining and refactoring personal websites and web applications and other web stuff.

I enjoy coding websites and web applications from scratch using HTML, CSS, SASS, JavascriptES6, PHP, Ajax and MySQL but, I also enjoy using web technologies like React, VueJS and Gatsby. I also like developing custom WordPress themes, blocks and plugins.

I have used what I have learnt to help others I have connected with who share an interest programming and learnt a lot from them as well. I've contributed my coding skills to help carry out events like Digital Futures 2017 at Waterloo London, volunteering at WordCamp 2018 in London and attended online pair programming sessions working with others solving coding challenges and learning code.
