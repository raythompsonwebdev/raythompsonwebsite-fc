About Me

I have had an interest in programming and web development since 2012 after attending a part-time web design course in Hackney where I learnt basic HTML, CSS and Javascript, Website Design and Website Development.

After course ended I went back to full-time work in Local Government but I still found programming and the whole web development process fascinating and started spending lots of my spare time self learning programming and more advanced web development techniques. I read books and watch video tutorials. I got practice coding by completing online programming courses and coding challenges.

I also practiced web development by designing, constructing and deploying websites and web applications online. I have a fondness for coding websites from scratch with HTML, CSS, Javascript, PHP and MY SQL but I also enjoy creating web applications with WordPress, ReactJS, VueJS and GatsbyJS.

I have gained valuable knowledge and experience meeting others interested in programming and web development. I had the opportunity to contribute to offline events like Digital Futures 2017 in Waterloo London, volunteer at WordCamp 2018 in London and attend online pair programming sessions to solve coding challenges.

I have a goal to continue improving my programming skills and web development techniques. I would love able to contribute in the creation of well designed and maintainable web applications that help in the improvement of the user experience, security and accessibility of web applications.
