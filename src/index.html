About

I was inspired to start dabbling in web development in 2012 after attending a part-time web design course in my local area, where I learned basic Javascript, HTML, CSS, building websites. When I am not at my day job as a revenues officer, I have spent much of my free time coding, creating lots of web applications and keeping up to date with constant changes in field of web development.
My coding languages of choice are HTML, CSS, Javascript and PHP which I use along with development tools like React, VueJS, Node, Sass, customer management system WordPress and databases like Postgress aand MungoDB .
I enjoy solving coding challenges participate as much as I can in the web development community. I enjoy the challenge of implementing ideas from conception into fully working applications. I am enthusiastic about coding and web development and aspire to combine acquired skills and knowledge to produce great applications that provide all users with a great web experience.
