About

I was inspired to learn about web development in 2012 after attending a part-time web design course in my local area. I learned about HTML, CSS, Web Design, content creation and the process of building websites and web application. I spend much of my free time coding, creating web applications and keeping up to date with the frequent changes in web development. My coding languages of choice are Javascript and PHP which I like to use along with development frameworks and customer management systems like React, VueJS and WordPress.
